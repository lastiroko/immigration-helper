# Immigration Helper - All-in-One Platform for International Students in Germany

🎯 **Mission:** Simplify the complex German immigration process for international students

## 🏗️ Architecture

- **Backend:** Spring Boot 3.2 (Java 17) - REST API
- **Frontend Web:** React 18 + TypeScript + Tailwind CSS
- **Mobile:** Kotlin + Jetpack Compose (Android)
- **Database:** PostgreSQL (production), H2 (development)

- [Mobile Setup](./mobile-android/README.md)

## 📊 Features

### MVP (Weeks 1-6)
- ✅ Immigration office finder (location-based)
- ✅ Document checklist generator (visa-type specific)
- ✅ User authentication (JWT)
- ✅ Premium subscription (Stripe)
- ✅ Email notifications

### Future Phases
- Appointment booking integration
- Document upload & OCR
- Multi-language support (DE, EN, AR, ES, FR)
- University partnerships (B2B)

## 🛠️ Tech Stack

**Backend:**
- Spring Boot 3.2, Spring Security, Spring Data JPA
- PostgreSQL, Redis (caching)
- Stripe (payments), SendGrid (emails)
- Docker, GitHub Actions (CI/CD)

**Frontend:**
- React 18, TypeScript, Tailwind CSS
- Axios, React Query, React Router
- Google Maps API
- Vite (build tool)

**Mobile:**
- Kotlin, Jetpack Compose
- Retrofit, Room Database
- Google Maps SDK, Material 3
